# Script to convert MAC address format

Simple script to convert MAC address between formats:
> 01-23-45-67-89-AB | 01:23:45:67:89:AB | 0123456789AB | 0123.4567.89AB

* none - 0123456789AB
* dot - 0123.4567.89AB
* colon - 01:23:45:67:89:AB
* dash - 01-23-45-67-89-AB

## Technologies
* Python3

## Setup
> python mac_address_conversion.py <MAC_address_in_any_format> <format_to_convert>

Examples:
```sh
python mac_address_conversion.py 0123456789AB dash
python mac_address_conversion.py 0123456789AB dot
```
